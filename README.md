# Smart Macro Engine 🔥

**Intelligent Keyboard Macro System with Text Expansion**

*A powerful Python-based macro engine that automatically replaces keystrokes with predefined text outputs. Perfect for programmers, writers, and productivity enthusiasts.*

---

## 🚀 **Created by Imran**

### 🌟 **Key Features**
- **Smart Key Detection** - Single keys and complex sequences
- **Multiple Input Formats** - Flexible rule definitions
- **Custom Typing Speeds** - Per-character delay control
- **Non-Blocking Architecture** - Never interrupts your workflow
- **Real-time Processing** - Instant text expansion

### 💡 **Perfect For**
- Code snippet expansion
- Email templates
- Address autofill
- Common phrases
- Programming shortcuts

### ⚡ **Quick Setup**
```python
engine = SmartMacroEngine()
engine.add_rule(['i'], 'imran')
engine.add_rule(['e', 'm'], 'email@example.com')
```
